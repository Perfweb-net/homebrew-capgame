# homebrew-capgame

Tap Homebrew personnel pour [CapGame](https://github.com/Perfweb-net/record).

## Installation

```bash
brew tap perfweb-net/capgame
brew install --cask capgame
```

## Statut

⚠️ CapGame n'est pas encore signé avec un certificat Apple Developer ID. Après l'installation, débloque l'app avec :

```bash
xattr -cr /Applications/CapGame.app
```
